# WallpaperX
WallpaperX is a simple android application built using <a href="https://developer.android.com/jetpack/compose">Jetpack compose - Androidâ€™s modern toolkit for building native UI</a>. It uses pixabay and unsplash api to fetch wallpapers.

### Technologies used
- Jetpack compose
- Dagger-Hilt
- Kotlin-flow
- Animated navigation from accompanist library
